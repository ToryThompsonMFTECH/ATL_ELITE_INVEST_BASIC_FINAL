# Images Directory

This folder contains all images for the Atlanta Elite Investment Properties website.

## Folder Structure

### `/testimonials/` - Client Testimonial Photos
Add profile photos for each testimonial. Expected filenames:
- `sarah-johnson.jpg` (or .png)
- `michael-chen.jpg`
- `emily-rodriguez.jpg`
- `james-williams.jpg`
- `patricia-davis.jpg`
- `robert-martinez.jpg`

**Recommended size:** 200x200px or larger (square format works best)
**Format:** JPG or PNG

### `/team/` - Team Member Photos
Add professional photos of team members. Expected filenames:
- `john-smith.jpg` (Founder & CEO)
- `sarah-thompson.jpg` (Operations Manager)
- `michael-rodriguez.jpg` (Property Specialist)

**Recommended size:** 400x400px or larger (square format works best)
**Format:** JPG or PNG

## Notes

- Images will automatically display once you add them with the correct filenames
- If an image is missing, a fallback avatar with the person's initial will be shown
- All images are optimized automatically by Next.js for performance
- Use high-quality images for best results
